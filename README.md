# Flipping Bot

Flipping Bot is a simple and responsive chatbot web application built with React and Vite. It allows users to send messages and receive automated chatbot responses in a clean chat-style interface.

## Live Demo

Visit the live project here:

https://saikat141.github.io/Flipping-Bot-/

## Features

* Clean and modern chatbot UI
* User and bot message bubbles
* Press Enter to send messages
* Prevents empty messages
* Responsive layout
* Built with React components
* Hosted using GitHub Pages

## Technologies Used

* React.js
* Vite
* JavaScript
* HTML5
* CSS3
* GitHub Pages
* supersimpledev chatbot package

## Installation and Setup

Clone the repository:

```bash
git clone https://github.com/Saikat141/Flipping-Bot-.git
```

Go to the project folder:

```bash
cd Flipping-Bot-
```

Install dependencies:

```bash
npm install
```

Run the project locally:

```bash
npm run dev
```

Open the local server link in your browser.

## Build for Production

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```
